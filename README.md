# TubeFlow GUI 

**TubeFlow GUI** is a modern, lightweight desktop application designed to make downloading YouTube content effortless. It wraps the industry-standard `yt-dlp` engine in a clean, user-friendly interface built with Python and Tkinter.

---

## Features

- **Multi-Format Support:** Download as **MP4**, **WebM**, or extract high-quality audio to **MP3**.
- **Quality Control:** Choose between **Best available**, **1080p**, or **720p**.
- **Playlist Management:** Toggle easily between single video downloads or entire playlists.
- **Live Terminal Console:** View real-time download logs, speeds, and ETAs directly within the app.
- **Visual Progress:** Integrated progress bar for quick tracking.
- **Custom UI:** Support for both **Dark** and **Light** modes.
- **Automatic Setup:** Automatically checks for and installs `yt-dlp` on first run.

---

## Installation & Usage

### 1. Requirements

* **Python 3.x**
* **FFmpeg** (Required for MP3 conversion and merging 1080p+ video).
* [Download FFmpeg here](https://ffmpeg.org/download.html)

### 2. Setup
* Clone this repository to your local machine:

      git clone [https://github.com/el-guemra-br/TubeFlow-GUI.git](https://github.com/el-guemra-br/TubeFlow-GUI.git)
  
      cd TubeFlow-GUI

### 3. Install Dependencies

    pip install yt-dlp 
    
### 4. Run the Application

    python TubeFlow-GUI.py
